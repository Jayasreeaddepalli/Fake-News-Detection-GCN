# Fake News Detection using Graph Convolutional Networks (GCN)

## 📌 Overview
This project detects fake news articles using Graph Convolutional Networks (GCN).  
It leverages embeddings, labels, and graph structures to classify news as real or fake.

## 📂 Dataset
- `fake_news_content.json` → Fake news samples  
- `real_news_content.json` → Real news samples  
- Preprocessed embeddings and labels stored in `.pt` files
